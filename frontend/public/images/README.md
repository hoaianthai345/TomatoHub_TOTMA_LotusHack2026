Frontend Images

Purpose
- Store static images used by the Next.js frontend.

How to use
- Put image files inside this folder or its subfolders.
- Reference them in the app with paths like `/images/your-file.png`.

Suggested structure
- `frontend/public/images/branding`
- `frontend/public/images/landing`
- `frontend/public/images/campaigns`
- `frontend/public/images/avatars`
